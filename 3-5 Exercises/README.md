# 3.5 Exercise

## File Locations
- `Input01.java` is located at `src/Input01.java`
- `Input03.java` is located at `src/Input03.java`

## Input01.java
Using `JOptionPane` dialogs to collect user input.
- Prompts the user to type something and prints the input.
- Converts the input to an integer and prints the value plus 1.
- Demonstrates parsing and printing in a single line using a dialog input.

## Input03.java
Using `Scanner` to read input from the console.
- Reads three integers entered by the user.
- Calculates and prints the sum of the three numbers.
- Closes the `Scanner` at the end of the program.
