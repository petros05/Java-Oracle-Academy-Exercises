# 3.2 Exercises

## Exercise Files

The exercises are located in the `src/` folder of this project.

1. **Chickens01.java**
- **Location:** `src/Chickens01.java`
- **Goal:** Calculates total eggs produced over Monday, Tuesday, and Wednesday using a reusable method (`countEggs`) with different input values.

2. **Chickens02.java**
- **Location:** `src/Chickens02.java`
- **Goal:** Tracks egg totals for three days, then computes daily average, estimated monthly average, and monthly profit.
